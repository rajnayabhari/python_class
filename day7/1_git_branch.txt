1. git branch
 => Displays all the branches and highlights the current branch

2. git branch <branch_name>
 => creates a new branch with the name provided

3. git checkout <branch_name>
 => Leaves the current branch and jumps the branch provided

4. git checkout -b <branch_name>
 =>creates a new branch and jumps to that branch by copying
  all the codes from current branch
  
  
5. git merge
 => Merges the code from the branch provided to the current
 branch
